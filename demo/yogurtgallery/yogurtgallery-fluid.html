YogurtGallery DemoFluid Layout Configuration

This is the demonstration page for YogurtGallery, a responsive JavaScript web image gallery.

The gallery below uses a "fluid" layout. Rows do not have a set number of items and flow based on the gallery parameters and the parent container's width. Try resizing this container to see how the items reflow at different widths.

For a demonstration of the "fixed" layout, see this example instead.
