YogurtGallery DemoFixed Layout Configuration

This is the demonstration page for YogurtGallery, a responsive JavaScript web image gallery.

The gallery below uses a "fixed" layout. Rows have a specified number of items (5 in this example), and item dimensions scale based on the parent container's width. Try resizing this container to see how the items reflow at different widths.

For a demonstration of the "fluid" layout, see this example instead.
